of the corresponding file is given to give a general overview of the overal architecture.

	> atlas_scr::coordinate_system --> seems to be unused.
		- C++ Lib

	> atlas_scr::parser
		- parser.h: 	definition of classes Parser, ClockModel and ClockCorrection
		- parser.cpp: 	implementation of class and class functions from parser.h

		- logger.h: 	definition of class Logger
		- logger.cpp: 	implementation of class and class functions from logger.h

		- reporter.h: 	definition of class Reporter
		- reporter.cpp: implementation of class and class functions from reporter.h
